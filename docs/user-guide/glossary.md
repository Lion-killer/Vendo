# Словничок

- **Синхронізація** — обмін даними з сервером: відправка твоїх замовлень і отримання свіжих.
- **Онлайн / Офлайн** — є / немає зв'язку із сервером.
- **Черга** — твої дії (замовлення / правки / видалення), що чекають відправки.
- **Статус замовлення:** Нове (чернетка) · Відправлено · Проведено (оброблене, лише перегляд) · Видалено (помічене на видалення).
- **Помітка видалення** — замовлення позначене на видалення, але ще можна відновити.
- **Конфлікт** — те саме замовлення змінили і на сервері, і в тебе; треба обрати версію.
