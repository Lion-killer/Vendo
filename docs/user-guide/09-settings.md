# 9. Налаштування

> **Коли це потрібно:** змінити вигляд/мову, очистити дані або вийти.

Меню профілю (аватар на головній): `[скріншот: меню профілю]`
- **Тема** (світла / темна).
- **Мова** (українська / російська / англійська).
- **Очистити дані** — прибирає кеш і чернетки, **авторизація лишається**. Якщо є невідправлені замовлення — попередить. `[скріншот: попередження очищення]`
- **Вийти** — повернутися до екрана входу.
