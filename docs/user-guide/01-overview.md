# 1. Огляд екрана

> **Коли це потрібно:** щоб орієнтуватися в додатку.

## Головний екран
`[скріншот: головна з підписами]`
- Виторг за сьогодні, кількість замовлень, середній чек, клієнтів.
- Кнопка **«Нове замовлення»**.
- Картка **«Синхронізація»** (час останньої).

## Верхня панель
- **Значок інтернету** (онлайн/офлайн). `[скріншот: онлайн vs офлайн]`
- **Кнопка синхронізації ⟳** — червона крапка = є невідправлені замовлення.
