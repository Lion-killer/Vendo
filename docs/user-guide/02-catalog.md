# 2. Каталог товарів

> **Коли це потрібно:** знайти товар і додати його до замовлення.

## Кроки
1. Відкрий **Каталог**. `[скріншот: каталог, групи]`
2. Заходь у групи/підгрупи або скористайся **пошуком**. `[скріншот: пошук]`
3. (Опційно) натисни **сканер штрихкодів** і піднеси до товару. `[скріншот: сканер]`
4. Відкрий картку товару — ціна, залишок, фото. `[скріншот: картка товару]`
5. Додай потрібну кількість до замовлення. `[скріншот: додавання]`

## Результат
Товар у поточному замовленні.

## Поради
- Фото підвантажуються у фоні; поки немає — три крапки (не помилка).
