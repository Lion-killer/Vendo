# 6. Робота без інтернету

> **Коли це потрібно:** немає зв'язку в полі.

## Що працює офлайн
- Перегляд каталогу/клієнтів/замовлень (з кешу).
- Створення, редагування, видалення, відновлення замовлень — складаються в **чергу**.

## Синхронізація
1. Коли з'явився інтернет — натисни **⟳** угорі (або картку «Синхронізація»). `[скріншот: кнопка синхронізації]`
2. Дочекайся повідомлення про результат (надіслано / помилки / конфлікти).

## Значки стану замовлення
`[скріншот: бейджі стану]`
- **Очікує** — у черзі на відправку.
- **Помилка / Конфлікт** — потребує уваги (див. розділ 7).

> Читання даних оновлюється у фоні само; **відправка черги — лише кнопкою**.
