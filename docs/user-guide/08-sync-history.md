# 8. Історія синхронізацій

> **Коли це потрібно:** перевірити, що саме надіслалось і що не пройшло.

## Кроки
1. Головна → меню профілю (аватар) → **Історія синхронізацій**. `[скріншот: меню профілю]`
2. Переглянь прогони: час + результат (надіслано / конфлікт / помилка). `[скріншот: список прогонів]`
3. Розгорни прогін, щоб побачити кожне замовлення. `[скріншот: розгорнутий прогін]`
4. З проблемного рядка — **перейти в замовлення** для вирішення. `[скріншот: перехід у замовлення]`

## Додатково
- Фільтр **«Лише з помилками»**.
- **Очистити історію**.
