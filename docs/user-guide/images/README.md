# Скріншоти посібника

Зробіть знімки на пристрої (мова інтерфейсу = мова посібника) і покладіть сюди. У текстах посібника мітки `[скріншот: …]` замініть на `![опис](images/файл.png)`.

> Терміни в посібнику вже звірені з інтерфейсом (підписи кнопок/статусів збігаються). Лишилось додати самі зображення.

## Чек-лист (за розділами)

**0. Вхід**
- [ ] екран входу зі сканером QR
- [ ] наведення на QR
- [ ] успішний вхід

**1. Огляд**
- [ ] головний екран із підписами елементів
- [ ] онлайн vs офлайн (значок угорі)
- [ ] нижня панель навігації
- [ ] меню профілю (аватар)

**2. Каталог**
- [ ] каталог із групами
- [ ] пошук
- [ ] сканер штрихкодів
- [ ] товар на весь екран (назва/артикул/штрихкод/ціна/залишок)
- [ ] додавання товару

**3. Клієнти**
- [ ] список клієнтів
- [ ] пошук
- [ ] фільтр боргу
- [ ] картка клієнта

**4. Оформлення**
- [ ] каталог із доданим товаром і лічильником кошика
- [ ] вибір клієнта в замовленні
- [ ] екран замовлення з кнопкою «Зберегти»
- [ ] список із замовленням «очікує» + кнопка синхронізації

**5. Мої замовлення**
- [ ] список замовлень + фільтр періоду
- [ ] бейджі статусів
- [ ] меню дій у замовленні

**6. Офлайн**
- [ ] кнопка синхронізації з червоною крапкою
- [ ] бейджі стану на замовленнях

**7. Конфлікти**
- [ ] банер конфлікту з кнопками
- [ ] повідомлення про проведене
- [ ] діалог зняття помітки

**8. Історія синхронізацій**
- [ ] список прогонів
- [ ] розгорнутий прогін
- [ ] перехід у замовлення

**9. Налаштування**
- [ ] попередження про очищення з невідправленими

**10. Усунення проблем**
- [ ] екран «Сталася помилка»
- [ ] журнал помилок
