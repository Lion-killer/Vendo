# Скріншоти посібника

Знімки зроблено автоматично (headless-браузер, демо-дані, мобільний вʼюпорт, світла тема). Мова інтерфейсу = мова посібника.

## ✅ Готові (вбудовані в текст)
`login` · `dashboard` · `profile-menu` · `catalog` · `customers` · `orders` (зі статусами/бейджами) · `conflict-banner` · `sync-history` · `sync-history-expanded` · `clear-data-warning`

Покривають усі головні екрани та ключові стани (черга/конфлікт/історія/очищення).

## ⬜ Лишилось зняти на пристрої (деталі/зуми)
Це детальні/контекстні кадри, які важко відтворити в демо:
- наведення на QR · успішний вхід
- пошук (каталог/клієнти) · **сканер штрихкодів** (нативний) · товар на весь екран · додавання товару
- картка клієнта · фільтр боргу
- вибір клієнта в замовленні · екран замовлення з «Зберегти» · меню дій
- повідомлення «проведене не можна редагувати» · діалог зняття помітки
- перехід у замовлення з історії
- екран «Сталася помилка» · журнал помилок

> Замінюй мітку `[скріншот: …]` на `![опис](images/файл.png)` після додавання файлу.
