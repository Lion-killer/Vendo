## ⬜ Лишилось зняти на пристрої
Кадри, які залежать від нативних можливостей або станів сервера й не відтворюються в демо-браузері:
- **сканер штрихкодів** — нативна камера
- наведення на QR · успішний вхід — нативна камера
- екран «Сталася помилка» — аварійний стан
- повідомлення «проведене не можна редагувати» · діалог зняття помітки — потребують відповідного стану в 1С
- перехід у замовлення з історії — навігаційний перехід
