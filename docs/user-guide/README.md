## Шпаргалка (одна сторінка)
1. **Вхід:** відскануй QR від адміністратора → готово.
2. **Замовлення:** Каталог → додай товари → Клієнт → **Відправити**.
3. **Синхронізація:** натисни значок ⟳ угорі, коли є інтернет (червона крапка = є невідправлене).
4. **Без інтернету** все працює — замовлення чекають у черзі й підуть після синхронізації.
