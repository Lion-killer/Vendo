# 10. Якщо щось зламалось

> **Коли це потрібно:** додаток поводиться дивно або показав екран помилки.

## Екран помилки
`[скріншот: екран «Сталася помилка»]`
- **Надіслати лог розробнику** — відкриє «Поділитися» (надішли файл у месенджер/пошту).
- **Перезапустити додаток**.

## Надіслати лог вручну
Меню профілю → **Журнал помилок** → «Надіслати лог розробнику». `[скріншот: журнал помилок]`

## Типові ситуації
- **Порожній екран замість даних** — немає зв'язку або сервер відповідає помилкою; угорі з'явиться банер. Спробуй пізніше або надішли лог.
- **Замовлення «очікує» довго** — натисни синхронізацію за наявності інтернету.
