# 4. Оформлення замовлення

> **Коли це потрібно:** створити нове замовлення клієнту.

## Кроки
1. **Нове замовлення** (з головної) або Каталог → додай товари. `[скріншот: кошик]`
2. Признач **клієнта**. `[скріншот: вибір клієнта]`
3. Перевір кількості й суму. `[скріншот: позиції та сума]`
4. **Відправити** (онлайн) або зберегти — піде в чергу (офлайн). `[скріншот: кнопки]`

## Результат
- Онлайн: замовлення відправлене (статус «Відправлено»).
- Офлайн: збережене локально, чекає синхронізації (значок «очікує»).

## Поради
- Чернетка зберігається автоматично при виході з екрана.
- «Скопіювати в нове» — повторне замовлення тим самим клієнтом.
