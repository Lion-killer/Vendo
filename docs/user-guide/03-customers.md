# 3. Клієнти

> **Коли це потрібно:** обрати клієнта або переглянути його дані/борг.

## Кроки
1. Відкрий **Клієнти**. `[скріншот: список клієнтів]`
2. Знайди клієнта пошуком (назва, код, адреса, телефон). `[скріншот: пошук]`
3. (Опційно) фільтр **«З боргом»**. `[скріншот: фільтр боргу]`
4. Відкрий картку — адреса, телефони, контактні особи, борг. `[скріншот: картка клієнта]`

## Результат
Бачиш дані клієнта; можеш обрати його для замовлення.
