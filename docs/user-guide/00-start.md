# 0. Початок роботи (вхід)

> **Коли це потрібно:** перший запуск додатку або після виходу з акаунта.

## Що дасть адміністратор
- QR-код пристрою (містить адресу та код прив'язки), або
- дані для входу окремо (ідентифікатор + код).

## Кроки
1. Відкрий додаток. `[скріншот: екран входу зі сканером QR]`
2. Наведи камеру на QR-код. `[скріншот: наведення на QR]`
3. Зачекай повідомлення про успішний вхід. `[скріншот: успішний вхід]`

## Результат
Відкривається головний екран; дані каталогу/клієнтів завантажуються.

## Якщо не вийшло
- Перевір інтернет.
- Якщо код «невірний або використаний» — попроси адміністратора згенерувати новий.
