# 5. Мої замовлення

> **Коли це потрібно:** переглянути, відредагувати, видалити чи відновити замовлення.

## Як відкрити
Нижня навігація → **Замовлення**. Угорі обери період: швидкі пресети (сьогодні, 7 днів…) або **свій період**.

![Замовлення зі статусами](images/orders.png)

## Статуси замовлення
(бейджі статусів видно на скріншоті вище)
- **Нове** — чернетка на пристрої, ще не відправлена в офіс (має позначку **очікує**).
- **Відправлено** — потрапило в офіс; можна редагувати.
- **Проведено** — оброблене в офісі; **лише перегляд** (редагувати/видаляти не можна).
- **Видалено** — помічене на видалення; можна **відновити**.

> Нове замовлення лишається **Нове / очікує**, доки не натиснеш **Синхронізацію** — тоді стане **Відправлено**.

## Дії з замовленням
Відкрий замовлення → меню (⋯): `[скріншот: меню дій у замовленні]`
- **Редагувати** товари/кількість (для «Відправлено»).
- **Видалити** — для «Нове» прибирає чернетку; для «Відправлено» ставить помітку видалення.
- **Зняти помітку видалення** — повернути «Видалено» назад у роботу.
- **Скопіювати в нове** — повторити замовлення тим самим клієнтом.

## Результат
Зміни зберігаються в додатку й відправляються в офіс під час наступної синхронізації.
